### Visual Design

As our platform is designed around badminton events and community interaction, through thorough discussion we have decided that the interface needed to feel energetic, modern, and easy to navigate while still remaining clear and accessible. The visual direction was influenced by sports and event based platforms that use strong typography, bold layouts, and card based interfaces to support fast navigation and content discovery.

<img src="assets/images/moodboard.png" alt="Moodboard" width="90%">

**Figure 1. Moodboard exploring sports, events and community focused interface inspirations**

This moodboard helped identify several common visual patterns that influenced the platform’s direction. Many references used high contrast layouts, large imagery, and strong hierarchy to guide users towards important interactions. This aligned well with the platform’s focus on events and quick user engagement.

Additionally, several references balanced community interaction with structured layouts, which helped influence the decision to combine forum interactions, event discovery, and maps within a consistent interface system.

### Interface Design

Once we had a clearer vision of our visual direction, we selected a colour palette that could support readability, interaction clarity and a more energetic sports focused identity.

<img src="assets/images/colours.png" alt="Colour Palette" width="90%">

**Figure 2. Colour palette used to establish hierarchy, readability and clarity**

Blue was selected as the primary interaction colour because it creates strong visibility for buttons and call-to-action such as RSVP interactions. Since the event system is the platform’s main feature, making interaction elements highly visible became an important usability consideration.

Orange and lime green were used more selectively for highlights, skill-level tags, and smaller interaction elements. These colours help create stronger contrast throughout the interface and make important information easier to identify quickly.

Neutral colours such as white, dark grey, and black were also important in balancing the interface. Since the platform contains large amounts of information such as event details, posts, and maps, the neutral backgrounds help reduce visual clutter and improve readability.

### Accessibility

Accessibility was also an important consideration throughout the visual design process, especially because the brief requires the platform to meet AA accessibility standards.

Several design decisions were made to support readability and usability, including:
- High contrast between text and backgrounds  
- Clear button visibility  
- Consistent spacing and layout structure  
- Large and readable typography  
- Visible interaction states and tags 

The interface also avoids overly decorative elements that could distract from important interactions. Instead, the design focuses on clarity and usability so users can quickly scan information such as event details, skill levels, and RSVP actions.

This became especially important for the events system, where users need to quickly identify relevant sessions and interact with the platform efficiently.

#### Interface Consistency

Another important design consideration involved maintaining consistency across different sections of the platform. Although the platform contains multiple systems such as forums, maps, profiles, and events, repeated interaction patterns help create a more predictable user experience.

Examples of this include:
- Consistent button styles  
- Repeated card layouts  
- Similar spacing and hierarchy across pages  
- Shared colour usage for interaction elements  

This consistency helps reduce confusion and improves navigation throughout the platform. It also strengthens the overall visual identity by making the different sections feel connected rather than separate systems.

<img src="assets/images/design-systems.png" alt="Design Systems" width="90%">

**Figure 3. Design Systems showing typography hierarchy and reusable interface components used throughout the platform**

This is the main design systems we used across our entire platform. This also supported accessibility by improving readability and ensuring that important information could be more easily identified across different sections of the platform.

Overall, visual design is not only about aesthetics, but also about supporting usability and interaction. The moodboard and colour palette helped establish a more energetic and sports focused identity while still maintaining readability and accessibility.

By creating a consistent visual system across the platform, the interface became more focused and easier to navigate, helping support the platform’s overall goal of creating a more engaging badminton community experience.
